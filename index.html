Callables

fork

Creates a copy of the calling process.
Takes no arguments.
Returns child pid for the parent process or 0 for the child.

wait

Blocks until a child process is terminated and reaps it.
Takes pid as argument or -1 to wait for any child.
Returns 0 on success or -1 on error.

yield

Suspends the calling process if there are any waiting processes.
Takes no arguments.
Returns 0 on success.

kill

Attempts to terminate a process.
Takes pid as argument.
Returns 0 on success or -1 on error.
